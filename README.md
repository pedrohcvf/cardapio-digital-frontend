# Cardápio Digital - Frontend

Interface web para gerenciamento de cardápio digital, consumindo a [Cardápio Digital API](link-do-repo-do-back).

## Tecnologias

- React 18
- TypeScript
- TanStack Query (React Query)
- Axios
- Vite

## Funcionalidades

- Listagem de itens do cardápio
- Cadastro de novos itens via modal
- Integração com API REST

## Como rodar

1. Certifique-se de que a [API](link-do-repo-do-back) está rodando em `localhost:8080`
2. Instale as dependências:
```bash
   npm install
```
3. Rode a aplicação:
```bash
   npm run dev
```
4. Acesse `http://localhost:5173`
